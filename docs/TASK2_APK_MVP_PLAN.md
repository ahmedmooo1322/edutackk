# Task 2 APK MVP Plan

## Goal

Replace Telegram user experience with an Android APK that uses the Task 1 backend.

## Milestone 2.1 included in this package

- Student login
- Student register
- Backend URL setting
- Subscription status screen
- Student AI chat
- Job polling
- Logout

## Later Task 2 milestones

### 2.2 Quiz screen

- Get pending quiz
- Answer quiz
- Poll grading job
- Show feedback

### 2.3 Teacher APK screens

- Login as teacher
- My students
- Send quiz
- View progress

### 2.4 Parent APK screens

- Login as parent
- My children
- Last report
- Ask AI about child

### 2.5 Production polish

- Better Arabic UI
- Push notifications
- App icon/splash
- Offline retry
- Error tracking
- Release signing


## Live backend update

The APK default API base URL is set to `https://ai.elbahet-sm.com`. The Settings screen can still override it for local testing.
